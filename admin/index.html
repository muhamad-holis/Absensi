📖 Panduan Setup Google Form + Sheets

🎯 Tujuan Setup Ini

Karyawan absen lewat Google Form dari HP masing-masing → data otomatis masuk ke Google Sheets → admin lihat semua data di dashboard ini secara real-time.

BAGIAN 1 — Buat Google Form

1

Buka Google Form

Buka browser → ketik forms.google.com → login akun Gmail admin → klik "+" untuk buat form baru.

2

Atur Judul Form

Klik "Formulir Tanpa Judul" → ganti dengan: 📋 ABSENSI KARYAWAN HARIAN LEPAS

3

Tambah Pertanyaan 1 — Nama Karyawan

Klik "+" → pilih tipe Dropdown → isi nama karyawan satu per satu → centang "Wajib diisi".
Tip: Setiap ada karyawan baru, tambahkan nama di sini.

4

Tambah Pertanyaan 2 — Tanggal Absen

Klik "+" → pilih tipe Tanggal (Date) → centang "Wajib diisi".

5

Tambah Pertanyaan 3 — Status Kehadiran

Klik "+" → pilih tipe Pilihan Ganda → isi opsi: Hadir, Tidak Hadir, Izin → centang "Wajib diisi".

6

Tambah Pertanyaan 4 — Jam Masuk

Klik "+" → pilih tipe Waktu (Time) → centang "Wajib diisi".

7

Tambah Pertanyaan 5 — Jam Keluar

Klik "+" → pilih tipe Waktu (Time) → tidak perlu wajib diisi.

8

Tambah Pertanyaan 6 — Foto Selfie

Klik "+" → pilih tipe Upload File → atur: izinkan Gambar (Image), ukuran maks 10 MB → centang "Wajib diisi".

9

Tambah Pertanyaan 7 — Catatan (Opsional)

Klik "+" → pilih tipe Jawaban Singkat → tidak perlu wajib diisi. Untuk keterangan izin, sakit, dll.

BAGIAN 2 — Hubungkan ke Google Sheets

10

Buka Tab Respons

Di halaman Google Form → klik tab "Respons" di bagian atas form.

11

Buat Google Sheets

Klik ikon Google Sheets 📊 → pilih "Buat spreadsheet baru" → beri nama REKAP ABSENSI KARYAWAN → klik Buat.

12

Atur Akses Sheets (PENTING!)

Di Google Sheets → klik tombol Share / Bagikan → klik "Change to anyone with the link" → pastikan role: Viewer → klik Done.

13

Copy Link Sheets

Copy link Google Sheets dari address bar browser → paste di tab "🔗 Koneksi Sheets" di dashboard ini → klik Hubungkan.

BAGIAN 3 — Bagikan Form ke Karyawan

14

Dapatkan Link Form

Di Google Form → klik tombol Kirim (kanan atas) → klik ikon link 🔗 → centang "Persingkat URL" → klik Salin.

15

Kirim ke Karyawan

Bagikan link lewat WhatsApp grup → minta karyawan simpan link (bookmark di HP). Setiap hari kerja buka link → isi form → kirim.

16

Buat QR Code (Opsional)

Buka qr-code-generator.com → paste link form → download QR Code → print dan tempel di lokasi kerja. Karyawan cukup scan kamera HP!

⚡ Setelah semua setup selesai:

Karyawan isi form → data muncul otomatis di Google Sheets → klik "🔄 Sync Sekarang" di tab Koneksi Sheets → semua data karyawan tampil di dashboard ini untuk rekap, laporan, dan slip gaji.
