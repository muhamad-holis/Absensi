Cara Setup Google Apps Script

1

Buka Google Apps Script

Buka browser → ketik script.google.com → login Gmail → klik "+ New Project"

2

Hapus kode default & paste kode Apps Script

Hapus semua kode yang ada → copy kode dari file apps-script.gs yang sudah disediakan → paste di editor → klik 💾 Save

3

Deploy sebagai Web App

Klik Deploy → New Deployment → pilih tipe: Web App
→ Execute as: Me
→ Who has access: Anyone
→ Klik Deploy → izinkan akses

4

Copy URL Deployment

Setelah deploy, akan muncul URL seperti: https://script.google.com/macros/s/XXXXX/exec
→ Copy URL tersebut → paste di kolom di atas → klik Hubungkan

5

Copy URL ke Halaman Absen Karyawan

URL yang sama juga di-paste di halaman absen-karyawan-online.html agar karyawan bisa absen dari HP masing-masing.

⚠️ Jika muncul "Authorization Required":

Saat pertama deploy, Google akan minta izin. Klik "Review Permissions" → pilih akun Gmail Anda → klik "Advanced" → klik "Go to (nama project)" → klik "Allow". Ini normal dan aman.

📋 File yang sudah disediakan:

• apps-script.gs — kode untuk di-paste ke Google Apps Script
• admin-dashboard.html — halaman ini (dashboard admin)
• absen-karyawan-online.html — halaman absen untuk karyawan
